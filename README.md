# 🎋 Bamburiti - Ecossistema Sustentável

![Status do Projeto](https://img.shields.io/badge/status-em%20desenvolvimento-green)
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![Styled Components](https://img.shields.io/badge/styled--components-DB7093?style=for-the-badge&logo=styled-components&logoColor=white)

O **Bamburiti** é uma plataforma web dedicada à promoção de um estilo de vida sustentável, focada no uso do bambu e na mobilidade urbana consciente. O projeto une design moderno, responsividade e uma interface intuitiva para conectar usuários a soluções ecológicas.

---

## 🚀 Funcionalidades Implementadas

* **Design Responsivo:** Interface adaptável para dispositivos móveis (smartphone) e desktop.
* **Menu Hambúrguer:** Navegação fluida e moderna para versões mobile.
* **Ecossistema Digital:** Componentes organizados para exibição de vantagens, bikes e informações institucionais.
* **Módulo de Registro:** Layout otimizado para cadastro de usuários interessados no ecossistema.
* **ChatBot:** Assistente virtual integrado para suporte rápido ao usuário.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento Front-End:

* **[React.js](https://reactjs.org/):** Biblioteca principal para construção da interface.
* **CSS3 & Flexbox:** Para um layout moderno e responsividade total.
* **JavaScript (ES6+):** Lógica de componentes e interatividade.
* **Git & GitHub:** Versionamento de código e gerenciamento de repositório.

---

## 📱 Visualização Mobile

O projeto foi refatorado para garantir que a experiência do usuário seja perfeita em telas menores, utilizando:
* Media Queries estratégicas.
* Componentização de elementos repetitivos.
* Otimização de imagens (WebP) para carregamento rápido.
